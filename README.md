## 1. 设计概述

### 设计基调

| 维度 | 说明 |
|------|------|
| **风格** | 温柔治愈的轻氛围感，简约不杂乱 |
| **配色** | 暖白 + 奶茶棕（`#FEFAF6` / `#C4956A` / `#4A3728`） |

| **适配** | 移动端优先，响应式适配手机/平板/电脑 |

| 技术 | 用途 | 引入方式 |
|------|------|----------|
| Tailwind CSS | 页面布局与样式 | CDN 引入 |
| canvas-confetti | 彩带动效 | CDN 引入 |

## 3. 功能清单

### 基础功能

| 功能 | 说明 |
|------|------|
| 加载过渡页 | 打开页面先显示 2 秒柔和过渡动画，让资源加载完再展示 |

## 5. 如何启动预览

### 方式一：直接用浏览器打开（最简单）

```
1. 找到 f:/lifeCode/birthday/index.html
2. 双击用浏览器打开
```

> ⚠️ 直接打开时部分功能受限（Service Worker 不生效、PWA 不可用），但页面主体完全正常。  
> 如果要完整体验 PWA 和离线缓存，请用方式二。

### 方式二：本地服务器（完整体验）

#### 用 VS Code（推荐）

1. 安装 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 插件
2. 右键 `index.html` → "Open with Live Server"
3. 浏览器自动打开 `http://127.0.0.1:5500/`

#### 用 Node.js

```bash
cd f:/lifeCode/birthday
npx serve .
# 打开 http://localhost:3000
```

#### 用 Python

```bash
cd f:/lifeCode/birthday
python -m http.server 8080
# 打开 http://localhost:8080
```

### 方式三：手机预览

1. 电脑上启动本地服务器
2. 确保手机和电脑在同一 Wi-Fi
3. 查看电脑的局域网 IP（终端执行 `ipconfig`）
4. 手机浏览器访问 `http://你的电脑IP:端口号`

## 7. 如何部署上线

### 推荐方案：Gitee Pages（国内访问快，免费）

#### 步骤 1：创建 Gitee 仓库

1. 注册/登录 [Gitee](https://gitee.com)
2. 新建仓库，名称随意（如 `birthday`）
3. 仓库设为**公开**

#### 步骤 2：上传文件

```bash
# 方式一：网页上传
直接把 birthday/ 文件夹里的所有文件拖到 Gitee 仓库网页

# 方式二：Git 命令行
cd f:/lifeCode/birthday
git init
git add .
git commit -m "老婆30岁生日纪念页"
git remote add origin https://gitee.com/你的用户名/birthday.git
git push -u origin master
```

#### 步骤 3：开启 Pages 服务

1. 进入仓库 → **服务** → **Gitee Pages**
2. 部署分支：选择 `master`
3. 部署目录：留空（根目录）
4. 点击**启动**
5. 等待几秒钟，获得网址（格式：`https://你的用户名.gitee.io/birthday`）

#### 步骤 4：生成二维码

1. 复制 Gitee Pages 给你的网址
2. 打开 [草料二维码](https://cli.im/) 或其他二维码生成工具
3. 粘贴网址，生成二维码
4. 下载高清二维码图片，准备打印或制作卡片

### 备选方案：GitHub Pages

步骤同上，只是平台换成 GitHub。GitHub Pages 免费，但国内访问稍慢。

### 备选方案：EdgeOne Pages

在 CodeBuddy 中点击集成菜单 → EdgeOne Pages → 一键部署，非常快速方便。

## 9. 未来更新指南

这个页面设计为"可持续更新的线上纪念册"，以后每年都可以加新内容。

### 每年怎么更新

1. 打开 `index.html`
2. 新增照片：在 `images/` 文件夹加入新照片
3. 在时光轴中新增节点（比如结婚纪念日、宝宝周岁等）
4. 在照片墙中替换/新增日常照片
5. 更新"心里话"部分
6. 更新"未来留言板"的内容
7. 重新部署到 Gitee Pages（覆盖即可，同一个网址不会变）

### 可以扩展的内容（可选）

- **宝宝成长模块**：记录宝宝每年的变化
- **旅行纪念**：每次旅行的照片和回忆
- **年度回顾**：每年生日更新一段当年的小总结

## 10. 常见问题

### Q：照片显示不出来怎么办？
A：检查 `images/` 文件夹中的文件名是否和 HTML 中引用的一致。注意文件名大小写。

### Q：音乐在微信里不能自动播放？
A：这是正常现象，微信浏览器禁止自动播放。已经做了手动触发设计——点右下角音符按钮即可播放。

### Q：手机端时间轴的图片被截断了？
A：已经在 CSS 中设置 `max-height: 300px` + `object-fit: cover`，如果图片本身比例特殊，可以单独调整。

### Q：倒计时显示不对？
A：检查你的电脑/手机时间是否正确。倒计时基于设备本地时间计算。

### Q：二维码扫不出来？
A：确认 Gitee Pages 已成功部署，尝试直接在浏览器打开网址确认能访问，再生成二维码。

### Q：想修改配色怎么办？
A：在 `index.html` 开头 `tailwind.config` 中修改 `colors.warm` 的几个值即可：
- `DEFAULT`: 页面主背景色
- `dark`: 深色文字
- `accent`: 强调色
- `soft`: 浅色装饰
- `muted`: 灰色文字

### Q：九宫格想加减照片数量？
A：在 HTML 中搜索 `photoGrid`，复制或删除 `<div class="grid-item">` 即可。照片源数组会自动读取。
